# 🚀 VANTAGE-AI: AI-Driven Content Strategy Dashboard

VANTAGE-AI is an advanced dashboard for digital creators to analyze post performance, predict viral probability, and generate strategic growth reports using AI-powered insights.

---

## ✨ Key Features
* **Daily Analysis Report:** Deep dive into specific post performance across platforms.
* **Viral Probability Prediction:** AI-driven forecasting for content reach.
* **Weekly Strategic Overview:** Long-term trend analysis and growth suggestions.
* **Automated PDF Reports:** Generate and download professional strategic reports.
* **Autonomous AI Agent:** Uses FastAPI and Google Gemini to process audience doubts and content calendars.

---

## 🛠️ Project Structure
* **`app.py`**: Frontend Streamlit application for the dashboard UI.
* **`main.py`**: Backend FastAPI endpoint for autonomous AI tasks.
* **`requirements.txt`**: List of all necessary libraries.

---

## 🚀 Setup and Installation

1. **Install Dependencies:** `pip install -r requirements.txt`
2. **Run Backend (FastAPI):** `uvicorn main:app --reload`
3. **Run Frontend (Streamlit):** `streamlit run app.py`
